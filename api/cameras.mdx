## Overview

The Cameras API manages surveillance camera infrastructure including fixed cameras, domes, LPR cameras, and facial recognition cameras. Each camera is associated with a site and has properties for location, orientation, and technical specifications.

## Endpoints

### List Cameras

Retrieve a paginated list of surveillance cameras with statistics.

```http
GET /camaras
```

<ParamField query="page" type="integer" default="1">

### Create Camera

Create a new surveillance camera record.

```http
POST /camaras
```

**Request Body**

<ParamField body="tipo_camara_id" type="integer" required>
  Camera type ID (cannot be "Selecciona un tipo de cámara")
</ParamField>

<ParamField body="sitio_id" type="integer" required>
  Site ID (cannot be "Seleccionar")
</ParamField>

<ParamField body="ip" type="string">
  Camera IP address
</ParamField>

<ParamField body="nombre" type="string">
  Camera name or description
</ParamField>

<ParamField body="orientacion" type="string">
  Camera orientation (NORTE, SUR, ESTE, OESTE, SURESTE, SUROESTE, NORESTE, NOROESTE)
</ParamField>

<ParamField body="angulo" type="decimal">
  Viewing angle in degrees
</ParamField>

<ParamField body="inteligencia" type="string">
  Intelligent features description
</ParamField>

<ParamField body="nro_serie" type="string">
  Serial number
</ParamField>

<ParamField body="fecha_instalacion" type="date">
  Installation date
</ParamField>

<ParamField body="etapa" type="string">
  Project stage/phase
</ParamField>

<ParamField body="observaciones" type="text">
  Observations
</ParamField>

**Example Request**

```bash
curl -X POST "https://your-domain.com/camaras" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -H "Cookie: your-session-cookie" \
  -d "tipo_camara_id=3" \
  -d "sitio_id=12" \
  -d "ip=192.168.1.100" \
  -d "nombre=Camara Entrada Principal" \
  -d "orientacion=NORTE" \
  -d "angulo=45" \
  -d "fecha_instalacion=2024-01-15"
```

**Response**

Redirects to `/camaras` on success.

**Note**: Latitude and longitude are automatically set from the site's location.

### Update Camera

Update an existing camera record.

```http
PUT /camaras/{id}
```

<ParamField path="id" type="integer" required>
  Camera ID to update
</ParamField>

**Request Body**

Same parameters as create endpoint.

**Example Request**

```bash
curl -X PUT "https://your-domain.com/camaras/45" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -H "Cookie: your-session-cookie" \
  -d "tipo_camara_id=3" \
  -d "sitio_id=12" \
  -d "ip=192.168.1.101" \
  -d "observaciones=IP address updated"
```

**Response**

Redirects to `/camaras` on success.

**Error Response**

<ResponseField name="error" type="string">
  "Error al actualizar cámara." - Camera update failed
</ResponseField>

## Additional Endpoints

### Restart Camera

Send reboot command to a camera.

```http
POST /camaras/{id}/reiniciar
```

<ParamField path="id" type="integer" required>
  Camera ID to restart
</ParamField>

**Response**

Returns success message with URL to open in new tab:

```
http://{user}:{pass}@{ip}/cgi-bin/magicBox.cgi?action=reboot
```

**Note**: Uses credentials from environment variables `CAMARA_USER` and `CAMARA_PASS`.

**Example Request**

```bash
curl -X POST "https://your-domain.com/camaras/45/reiniciar" \
  -H "Cookie: your-session-cookie"
```

### Import Cameras from Excel

Bulk import cameras from Excel file.

```http
POST /import-camaras
```

<ParamField body="excel_file" type="file" required>
  Excel file containing camera data
</ParamField>

**Example Request**

```bash
curl -X POST "https://your-domain.com/import-camaras" \
  -H "Cookie: your-session-cookie" \
  -F "excel_file=@camaras.xlsx"
```

**Response**

```json
{
  "success": "Los datos se han importado correctamente"
}
```

### Export Cameras to Excel

Export all camera records to Excel.

```http
GET /export-camaras
```

**Response**

Downloads Excel file: `ListadoCamaras_{timestamp}.xlsx`

**Example Request**

```bash
curl -X GET "https://your-domain.com/export-camaras" \
  -H "Cookie: your-session-cookie" \
  -o camaras.xlsx
```

## Camera Types

Supported camera types:

- **Fija** - Fixed camera
- **Fija - FR** - Fixed camera with facial recognition
- **Fija - LPR** - Fixed camera with license plate recognition
- **Fija - LPR NV** - Fixed LPR camera (night vision)
- **Fija - LPR AV** - Fixed LPR camera (analytics vision)
- **Domo** - Dome camera (PTZ)
- **Domo Dual** - Dual dome camera
- **BDE (Totem)** - Totem/BDE camera

## Orientation Values

Valid orientation values:

- NORTE
- SUR
- ESTE
- OESTE
- SURESTE
- SUROESTE
- NORESTE
- NOROESTE
